Projeto para estudos e testes em programação WEB, tentei adicionar o máximo de recursos que lembrei, 
Cadastrar, Consultar, Atualizar, Excluir, uploads de imagens, tempo de execução no servidor e frontend, 
recursos de segurança xss, também uma função para se o usuário não ter uma senha de acesso, 
fica em modo somente consulta, senha inicial é 1010 e se precisar outra é só criar um novo hash base64
***
This project is for studies and tests in web programming. I tried to add as many features as I could remember:

Register, Consult, Update, Delete, image uploads, server and frontend execution time,
XSS security features, and a function that works in query-only mode if the user doesn't have an access password. The initial password is 1010, and if a different password is needed, a new base64 hash can be created.
